# api-bet365-node

- API para requisição de dados de jogos ao vivo, Macths e Jogos Futuros.

## instalação

```sh
~$ npm install api-bet365-node
```

## Utilização

```js
const bet356 = require('api-bet365-node');

var api = new bet356()

api.on().then((rest) => {
    console.log(rest)
});

```
